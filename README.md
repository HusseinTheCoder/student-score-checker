# Student Score Checker

Student Score Checker is a simple web project that allows users to enter student scores, calculate the average, determine the grade, and display whether the result is pass or fail.

## Project Description

This project was built as a beginner-friendly website to practice core front-end development skills using HTML, CSS, and JavaScript. It focuses on basic score processing logic and presenting the result clearly to the user.

The system takes score input, checks whether the values are valid, calculates the average score, assigns a grade based on the average, and then shows the final academic status.

## Features

- Input student scores
- Validate score values
- Calculate average score
- Assign grade automatically
- Show pass or fail result
- Simple and clean interface

## Grading Logic

The grading system used in this project is:

- A = 90 to 100
- B = 80 to 89
- C = 70 to 79
- D = 60 to 69
- F = below 60

A student passes if the average score is 60 or above.  
A student fails if the average score is below 60.
